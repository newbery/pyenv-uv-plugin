# Pyenv plugin: pyenv-uv (archived)

A [pyenv](https://github.com/pyenv/pyenv) plugin to support managing python
installs using [uv](https://docs.astral.sh/uv/reference/cli/#uv-python).

## Installation

This will install the latest development version of *pyenv-uv* into the
`$(pyenv root)/plugins/pyenv-link` directory.

### for `bash` shell:
```bash
git clone https://github.com/newbery/pyenv-uv-plugin.git $(pyenv root)/plugins/pyenv-uv
```

### for `fish` shell:
```fish
git clone https://github.com/newbery/pyenv-uv-plugin.git (pyenv root)/plugins/pyenv-uv
```
